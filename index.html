HTML

What is HTML?

Html stands for "hyper text markup language". It is used to define the layout of a webpage. It consists of a series of articles.

HTML Editors

Html program can be made in any basic text software.Software like notepad(in windows)and textedit(in mac) can be utilised for learning basic HTML.

HTML Document Structure

HTML document has a structure of declartion, heading, main container and body. Declaration defines the document being html. The main container defines the start and end of the code. Heading containg title defines the heading of the content and the body containing paragraphs contains the actual content of the program.

Tags

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

Types Of Tags

There are two types of tags.
1.Paired tags-These are the tags which require an opening and a closing tag. eg-p,b etc.
2.Unpaired Tags-These tags only require one tag(opening). eg- br,hr etc.

Some Common Tags

  1. Body- This tag defines the main content of the document and has attributes like background, background-color, text etc.
  2. P- This tag stands for paragraph and it defines the start and the end of paragraph.
  3. Br- This is a unpaired tag use to define line breaks.
  4. Hn- This is the heading tag in which n is a number defining the heading size h1 being the largest and h6 being the smallest.
  5. B, I and U- These are the tags used for formatting the text B to bold, I to italic and U to underline.
  6. SUB and SUP- These are the tags which are used to to make the text in SUB to subcript(Like This) and in SUP to superscript (Like this).
  7. FONT- This tag can be used to edit the font of a segment of a text. It's attributes are Size, color and face.
  8. TABLE- This tag is used to insert table in webpage. TD and TR are used to define the number of rows and columns in the table. It's attributes are boder, bordercolor, cellpadding and cellspacing.
